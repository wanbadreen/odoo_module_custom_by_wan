Loyalty Partial Redeem (Odoo 18)

This module enables partial redemption of loyalty points directly on Sales Orders. It integrates with Odoo’s native Loyalty Program, Loyalty Cards, and Loyalty History, so you don't need to maintain a separate loyalty engine.

Key Features

  • Redeem any number of loyalty points (supports partial redeem, not only full balance).
  • Automatic discount line creation on the quotation / sales order.
  • Configurable value per point (RM per point) in the wizard.
  • Automatic update of the loyalty card points after redeem.
  • Automatic creation of a loyalty history record for full traceability.
  • Simple, user-friendly wizard integrated into the standard Sales Order form.

How It Works

  1. Create a quotation or open an existing Sales Order.
  2. Click the Redeem Loyalty Points button in the header.
  3. A pop-up wizard shows the customer's available points.
  4. Enter how many points to redeem and confirm.
  5. The module computes the discount (points × RM per point) and adds a negative line to the order.
  6. The loyalty card balance and loyalty history are updated automatically.

Technical Details

  • Compatible with Odoo 18.
  • Depends on: sale_management, loyalty.
  • Wizard model: loyalty.partial.redeem.wizard.
  • Works with Odoo standard loyalty.program, loyalty.card, and loyalty.history.
  • Discount is applied using a dedicated service product (Loyalty Redemption).

Notes

This addon focuses on keeping the flow very simple for the sales team: open order → click button → redeem points → confirm. No complex configuration is required.
